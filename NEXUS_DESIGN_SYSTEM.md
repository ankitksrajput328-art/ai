# 🎨 Nexus AI: Premium Futuristic Design System

This system defines the visual language for a world-class AI ecosystem. It is built on the principles of **Glassmorphism**, **Neon Accents**, and **Immersive Motion**.

---

## 1. Color Palette (The "Midnight Galaxy" Palette)
- **Primary (Space Black):** `#050509` (Background)
- **Secondary (Eclipse):** `#0f172a` (Sidebar)
- **Accent (Electric Indigo):** `#6366f1` (Glows & Buttons)
- **Neon Pink (Power):** `#ec4899` (AI Thinking State)
- **Cyber Mint (Success):** `#10b981` (Online/Connected)
- **Glass Border:** `rgba(255, 255, 255, 0.08)`

---

## 2. Typography
- **Primary Font:** 'Outfit', sans-serif (Clean, Modern, Geometric)
- **Code Font:** 'JetBrains Mono' (High Legibility)
- **Scale:**
    - H1 (Hero): 48px / Bold
    - H2 (Section): 24px / Semi-Bold
    - Body: 15px / Regular
    - Caption: 12px / Medium

---

## 3. UI Components (The Glassmorphic Library)

### A. The "Nexus Card"
```css
.glass-card {
    background: rgba(255, 255, 255, 0.03);
    backdrop-filter: blur(20px);
    border: 1px solid rgba(255, 255, 255, 0.08);
    border-radius: 24px;
    box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.8);
}
```

### B. Neon AI Input
- Large, centered input field.
- Glowing border animation when "Deep Scan" is active.

### C. Chat Bubbles
- **User:** Subtle glass tint, right-aligned.
- **AI:** Indigo glow background, left-aligned with "Sparkle" icon.

---

## 4. Animation Ideas
1.  **Entrance:** Staggered fade-in for chat messages (Cubic-bezier).
2.  **AI Pulse:** A breathing neon glow around the logo when the AI is "Thinking".
3.  **Haptic Feedback:** Subtle scale transform (0.98) on button click.
4.  **View Switch:** Smooth slide-left/right transition when moving between Chat and Profile.

---

## 5. Screen Layouts (The Multi-View System)
- **Chat:** Focused, immersive, centered messages.
- **Tools Dashboard:** 3-column grid of glowing icons.
- **Profile:** Minimalist, high-contrast, summary of AI memory.
- **Pricing:** 3-tier cards with "Most Popular" glow effect.

---

**Architect's Goal:** The UI should feel like it was designed in the year 2030—clean, breathable, and alive.
